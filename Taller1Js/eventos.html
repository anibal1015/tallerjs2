Eventos JavaScript Comunes

Evento Descripción Ejemplo Aplicación
click Se activa cuando se hace clic en un elemento HTML.
mouseover Se activa cuando el puntero del mouse se mueve sobre un elemento HTML.
Pasar el mouse aquí
keydown Se activa cuando se presiona una tecla del teclado.
load Se activa cuando se carga completamente un elemento HTML (por ejemplo, una imagen o un documento).
submit Se activa cuando se envía un formulario HTML.
...
change Se activa cuando el valor de un elemento HTML cambia (por ejemplo, un campo de entrada).
mouseenter Se activa cuando el puntero del mouse entra en un elemento HTML.
Entrar aquí
mouseleave Se activa cuando el puntero del mouse sale de un elemento HTML.
Salir aquí
scroll Se activa cuando se desplaza el contenido de un elemento HTML.
Desplazarse aquí
focus Se activa cuando un elemento HTML recibe el foco (seleccionado para entrada de datos).
